Use Galaxy Note as digitizer or graphics tablet for PC

Demo

About

  • This was a small weekend project / the proof of concept for using Galaxy Note as a graphics tablet for PC
  • Input information is sent via UDP (by default if no IP address is specified data simply broadcasted)
  • It was tested in local network using WiFi
  • This app utilizes pen hover event to work more like usual digitizer tablet. Click down and up events happen when pen touches and is lifted off the screen
  • Works on Galaxy Note with ICS
  • At this point this solution only triggers mouse events without any pressure information. (hopefully I'll come with some ideas to emulate the pressure)
  • Finger touch, movements and hover only moves cursor without actualy drawing
  • There is a bug with orientation change so you must start S-Pen Digitizer in desired orientation

Installation

  1. Download Windows client (.exe (.Net application) for Windows and Android app for Galaxy Note)
  2. Install SPenDigitizer.apk on Galaxy Note
  3. Launch SPenClient.exe on Windows (Windows will ask you to allow access to Network - you should allow)
  4. Launch S-Pen Digitizer on your Note
  5. It should work by default, but for better performance you can change Host IP (in android app options) to point directly to your Windows machine
